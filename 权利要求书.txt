权利要求书

1. 一种基于多智能体辩论的无人机群综合性能评估方法，其特征在于，包括以下步骤：
步骤S1：轨迹抽象与证据表示，将原始无人机轨迹数据转换为结构化领域特定语言DSL，所述DSL保留关键运动学信息的同时将数据压缩约85%；
步骤S2：层次化多智能体辩论协议，根据信息论复杂度动态调整辩论深度，包括三层结构：快速共识层、深度分析层和元辩论层；
步骤S3：对抗-协作混合机制，采用动态红蓝队机制缓解回声室效应，红队和蓝队角色随轮次轮换；
步骤S4：动态角色轮换，防止视角锁定并确保全面的多方面分析；
步骤S5：结构化通信协议，智能体使用半结构化格式进行通信，每个响应包括主张、证据、反驳、摘要和置信度五个组件；
步骤S6：证据链可追溯性，确保每个主张由可验证的证据链支持，证据链包括原始数据、处理后的指标、推理步骤、主张和置信度分数；
步骤S7：多维共识建模，在分数、语义、优先级和关注点四个正交维度上量化共识；
步骤S8：元认知质量监控，计算证据质量、覆盖质量、连贯性质量和新颖性质量四个质量指标，如果总体质量低于阈值则触发主动干预；
步骤S9：确定性安全验证层，并行监控状态向量对抗硬约束，如果违反任何约束则注入否决信号；
步骤S10：最终评估报告生成，基于共识和监控结果生成最终评估报告。

2. 根据权利要求1所述的方法，其特征在于，所述步骤S1中轨迹抽象过程包括：
（1）分段：基于运动学属性使用滑动窗口方法对航向和速度进行变化点检测，将轨迹划分为逻辑段；
（2）事件提取：识别关键飞行事件，包括急转弯大于30度、快速高度变化大于3 m/s和接近警报；
（3）DSL生成：将分段数据使用标准化令牌格式化为紧凑文本块，标准化令牌包括SEG段细节、EVENT关键异常、ATTN需要注意的区域、WAYPTS航点偏差和SCORES指标摘要。

3. 根据权利要求1所述的方法，其特征在于，所述步骤S2中复杂度度量Hcomp结合事件密度和运动学熵：
Hcomp(𝒯) = α·|ℰ|/Tduration + β·Entropy(𝒱swarm)
其中，𝒯是轨迹段集合，ℰ是提取的关键事件集合，𝒱swarm是无人机群的速度分布，Tduration是任务持续时间，α和β是权重系数。

4. 根据权利要求1所述的方法，其特征在于，所述步骤S3中红队大小为：
Nred = max(1, ⌊N·ρred⌋)
其中ρred = 0.4，N是智能体总数；红队索引随轮次旋转：
ℐred(r) = {(i + r) mod N : 0 ≤ i < Nred}。

5. 根据权利要求1所述的方法，其特征在于，所述步骤S4中动态角色轮换包括：
（1）轮询轮换：ρRR(i, r) = a(i + r) mod N；
（2）自适应轮换：ρadapt(i, r, ℰ) = argmaxj∈[N] Relevance(aj, ℰ(r-1))，其中ℰ(r-1)是轮次r-1中出现的紧急问题，Relevance(·, ·)量化与识别问题的专业知识对齐。

6. 根据权利要求1所述的方法，其特征在于，所述步骤S7中多维共识建模包括：
（1）分数一致性：Cscore = 1 - std({γi : i ∈ [N]})，其中γi是智能体i的置信度；
（2）语义一致性：Csemantic = (2/(N(N-1)))∑i<j Sim(Si, Sj)，其中Si是智能体i的摘要，Sim(·, ·)计算序列相似性；
（3）优先级一致性：Cpriority = (2/(N(N-1)))∑i<j Sim(ci, cj)，其中ci是智能体i的主张；
（4）关注点一致性：Cconcern = |∩i Words(ei)|/|∪i Words(ei)|，其中ei是智能体i的证据。

7. 根据权利要求1所述的方法，其特征在于，所述步骤S8中总体质量分数是四个质量维度的最小值：
Q(r) = min{Qevidence(r), Qcoverage(r), Qcoherence(r), Qnovelty(r)}，如果Q(r) < τQ = 0.7，则触发主动干预。

8. 根据权利要求1所述的方法，其特征在于，所述步骤S9中否决信号为：
Signalveto(t) = ∨k 𝟙[ck(x(t)) > 0]，其中𝒞safe = { ck(x) ≤ 0 | k=1...K }是硬约束集合，x(t)是状态向量。

9. 根据权利要求1所述的方法，其特征在于，还包括专门化智能体角色定义，每个智能体初始化为特定核心价值和认知偏见定义的专门化角色：
（1）飞行控制专家FCS：设计为具有精确偏见，优先考虑定量指标而非定性行为，惩罚偏离标称轨迹大于5%的偏差，专注于机动的物理可行性；
（2）无人机群协调专家SCE：设计为具有凝聚力偏见，将无人机群评估为单一实体，惩罚即使其个体飞行路径安全也破坏编队的智能体，使用抑制个体异常数据而有利于群体统计的全局视图提示模板；
（3）安全评估专家SAE：设计为具有悲观偏见，在零信任模型下运作，将任何接近安全包络边界10%以内的状态向量标记为关键风险，指示在风险检测中优先考虑误报而非漏报。

10. 一种基于多智能体辩论的无人机群综合性能评估系统，其特征在于，包括：
轨迹抽象模块：用于将原始无人机轨迹数据转换为结构化领域特定语言；
层次化辩论协议模块：用于根据信息论复杂度动态调整辩论深度；
对抗-协作混合机制模块：用于采用动态红蓝队机制缓解回声室效应；
动态角色轮换模块：用于防止视角锁定并确保全面的多方面分析；
结构化通信协议模块：用于确保智能体使用严格的半结构化格式进行通信；
证据链可追溯性模块：用于确保每个主张由可验证的证据链支持；
多维共识建模模块：用于在四个正交维度上量化共识；
元认知质量监控模块：用于防止辩论退化为低质量的闲聊；
确定性安全验证模块：用于并行监控状态向量对抗硬约束；
最终评估报告生成模块：用于基于共识和监控结果生成最终评估报告。
